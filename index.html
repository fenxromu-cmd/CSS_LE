Exporter vers Outlook (.ics)

Choisissez ce que vous souhaitez exporter. Le fichier .ics généré peut être importé directement dans Outlook, Google Calendar ou Apple Calendar.

Comment importer dans Outlook :
1. Téléchargez le fichier .ics
2. Dans Outlook → Fichier → Ouvrir et exporter → Importer/Exporter
3. Choisissez "Importer un fichier iCalendar (.ics)"
4. Sélectionnez le fichier téléchargé
